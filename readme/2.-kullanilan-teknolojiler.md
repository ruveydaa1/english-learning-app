# 2. Kullanılan Teknolojiler

### Angular

Angular, uygulamanın kullanıcı arayüzünü geliştirmek için tercih edilmiştir. Bileşen tabanlı yapısı sayesinde modüler, ölçeklenebilir ve bakımı kolay bir frontend geliştirilmiştir.

#### Kullanım Amaçları

* Sayfa yönetimi
* Component yapısı
* HTTP istekleri
* Routing

***

### Node.js

Node.js, uygulamanın sunucu tarafında çalışmaktadır. Veritabanı işlemleri ve API servisleri Node.js ile geliştirilmiştir.

#### Kullanım Amaçları

* API geliştirme
* Veritabanı bağlantıları
* Sunucu yönetimi

***

### PostgreSQL

PostgreSQL, uygulamanın verilerini güvenli ve düzenli şekilde saklamak amacıyla kullanılmıştır.

#### Saklanan Veriler

* İngilizce kelimeler
* Türkçe karşılıkları

***

### API

Frontend ile backend arasındaki veri iletişimi REST API üzerinden gerçekleştirilmektedir.

API sayesinde;

* Kelime çekme API'si

işlemi güvenli şekilde gerçekleştirilmektedir.

***

#### Visual Studio Code

Visual Studio Code, projenin frontend ve backend geliştirme süreçlerinde kullanılan temel kod editörüdür.

**Kullanım Amaçları**

* Angular geliştirme
* Node.js geliştirme
* Kod düzenleme
* Hata ayıklama
* Terminal işlemleri

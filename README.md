# İngilizce Öğrenme Uygulaması
Bu proje, İngilizce kelime bilgisini geliştirmek ve dil öğrenme sürecini daha verimli, planlı ve eğlenceli hale getirmek amacıyla geliştirilmiş modern bir web uygulamasıdır.

## Proje Hakkında
Uygulama, kelime öğrenme, quiz çözme ve mini oyunlar gibi özellikleri tek bir platformda sunarak kullanıcıların farklı kaynaklara ihtiyaç duymadan öğrenme süreçlerini yönetmelerine olanak sağlar.

## Temel Özellikler
- **Kelime Kartları:** Yeni kelimeleri öğrenmeyi kolaylaştıran interaktif sistem.
- **Quiz Sistemi:** Öğrenilen bilgilerin pekiştirilmesini sağlayan değerlendirme modülü.
- **Mini Oyunlar:** Dil öğrenimini oyunlaştırarak daha keyifli hale getiren içerikler.
